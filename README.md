# CRC_MetaThermo_v1 code release

This repository contains the minimal reproducible code used for the manuscript:
multi-cohort single-cell analysis of colorectal cancer metastasis.

## Contents

- scripts/: figure generation and analysis scripts
- notebooks/: optional workflow notebooks, if included
- environment.yml: conda environment specification
- requirements.txt: pip package list

## Data

Raw datasets are publicly available from GEO:
GSE178318, GSE231559, GSE298084, GSE299737, and GSE110009.

Processed data are archived separately in the companion Zenodo data package.

## Reproducibility

The scripts are intended to reproduce principal figures and derived tables from the processed data package.
Internal platform code, API orchestration, server files, and private credentials are not included.
